这里展示了我在各种GameJam活动中参与的游戏作品

GameJam（或写作Game Jam）是一种游戏创作活动，通常在48小时至21天的有限时间内完成。参与者需要在规定主题下，快速组队、构思并实现一个可玩的游戏原型。它强调创意的迸发、团队的合作以及对游戏设计的探索。GameJam不仅是一次挑战时间与技能的体验，更是展示灵感、结识伙伴和突破自我的舞台。

林克传说 泡泡的再现

Global Game Jam 2025（48小时）

《林克传说 泡泡的再现》是一款卡牌策略类游戏，玩家需要合理安排攻击力浮动小的塞尔达和攻击力浮动大的米法，应对各种怪物或道具。

衔尾之蛇

CiGA Game Jam 2024（48小时）

《衔尾之蛇》是一款类贪吃蛇的解密游戏。玩家扮演一条能改变长度的“蛇”，通过改变自身的能力，去探索一座无限循环的迷宫。

Good Day

Global Game Jam 2024（48小时）

《Good Day》是一款图像化叙事的点击解密类游戏。玩家通过点击屏幕的方式操控主角行动，逐步了解主角忧郁的一天是如何度过的。
